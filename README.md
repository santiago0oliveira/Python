## Português
Contador de Palavras |
Um script simples em Python para contar o número total de palavras em um arquivo de texto e listar as 10 palavras mais frequentes.

---
📋 Descrição
Este script lê um arquivo de texto especificado pelo usuário, conta o número total de palavras e exibe as 10 palavras mais frequentes.

🚀 Como Executar
1. Certifique-se de ter o Python 3 instalado.
2. Execute o script no seu terminal:

Bash
    python contador_palavras.py

Digite o caminho para o seu arquivo .txt quando solicitado.

## English

# Word Counter | 
A simple Python script to count the total number of words in a text file and list the top 10 most frequent words.

---
### 📋 Description
This script reads a text file specified by the user, counts the total number of words, and displays the 10 most frequently used words.

### 🚀 How to Run
1. Make sure you have **Python 3** installed.
2. Run the script in your terminal:
Bash
   python contador_palavras.py

Enter the path to your .txt file when prompted.
